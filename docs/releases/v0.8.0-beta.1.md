# v0.8.0-beta.1

First beta milestone for the Notepad++ Linux fork.
Prepared on 2026-02-13.

## Release gate status

- Phase 9 complete (automatic language detection and syntax-highlighting quality gates).
- Linux local CI-equivalent lanes green (`debug` + `release` + ASan/UBSan local run).
- Release artifacts and checksums validated.

## Highlights

- Automatic language detection using extension, shebang, modelines, and content heuristics.
- Reliable Markdown/HTML/programming-language classification on open and rename/save-as.
- Confidence-scored auto-lexer switching with manual override lock.
- Detection accuracy/false-positive regression reporting.
