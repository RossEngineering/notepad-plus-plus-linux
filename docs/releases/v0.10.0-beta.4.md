# v0.10.0-beta.4

> **Linux-only fork notice:** This repository and its releases target Linux only. For the original Windows Notepad++ application, visit [notepad-plus-plus.org](https://notepad-plus-plus.org/).

Post-RC development maintenance prerelease for the Notepad++ Linux fork.
Prepared on 2026-02-14.
Published on: 2026-02-14.

## Release gate status

- Linux integration/distribution section close (`v0.10.0-beta.3`): complete.
- Packaging hotfix scope (`v0.10.0-beta.4`): complete.
- Linux CI + release packaging lanes: green for prerelease candidate commit.
- GitHub prerelease status: published.

## Scope for this prerelease

1. Fix RPM packaging path handling introduced in `v0.10.0-beta.3` release automation.
2. Keep native package outputs enabled and deterministic (`.pkg.tar.zst`, `.deb`, `.rpm`).
3. Publish maintenance prerelease without changing user-facing editor features.

## Highlights

1. RPM `%install` staging now copies into `%{buildroot}/usr` explicitly.
2. RPM `%files` now includes `/usr/share/notepad-plus-plus-linux/default-update-channel`.
3. Fedora RPM artifact generation succeeds for prerelease tags again.

## Known limitations

1. In-app update remains channel-selection only; auto-download/apply is still out of scope.
2. Wayland QA remains a smoke lane, not a full compositor matrix.
