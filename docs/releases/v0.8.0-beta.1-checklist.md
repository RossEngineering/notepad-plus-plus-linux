# v0.8.0-beta.1 Release Checklist

Use this checklist before creating and publishing the `v0.8.0-beta.1` tag.

## Scope freeze

- [ ] Beta 1 scope frozen to Phase 9 language detection/highlighting deliverables.
- [ ] Post-beta items (skinning/extensions) excluded from Beta 1 scope.

## Quality gate

- [ ] `language_detection_regression_test` passes with required thresholds.
- [ ] Core regression/smoke suite passes.
- [ ] Linux sanitizer CI lane passes.

## Packaging and artifacts

- [ ] Release artifacts generated (`.tar.xz`, `.sha256`, optional `.asc`).
- [ ] Checksums verified from generated files.
- [ ] Artifact names and version strings match `v0.8.0-beta.1`.

## Documentation

- [ ] `docs/releases/v0.8.0-beta.1.md` updated from draft to final release notes.
- [ ] `README.md` and `docs/roadmap.md` status updated for Beta 1 release state.
- [ ] Known limitations and post-beta backlog called out clearly.

## Publish

- [ ] `master` pushed to origin.
- [ ] Tag `v0.8.0-beta.1` created and pushed.
- [ ] GitHub Release published with matching artifacts.
