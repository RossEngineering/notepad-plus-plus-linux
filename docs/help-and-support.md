## Report a bug

Use the bug form:

- `https://github.com/RossEngineering/notepad-plus-plus-linux/issues/new?template=bug_report.yml`

Please include:

- Distro and version (for Arch derivatives, include the specific derivative).
- App version/tag or commit SHA.
- Repro steps and expected vs actual behavior.
- Logs/screenshots if available.

## Request a feature

Use the feature request form:

- `https://github.com/RossEngineering/notepad-plus-plus-linux/issues/new?template=2-feature-request.yml`

Please include:

- Problem statement and target workflow.
- Proposed behavior/UI.
- Any compatibility or performance constraints.
